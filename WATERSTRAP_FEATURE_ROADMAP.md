# Waterstrap Feature Roadmap

Waterstrap should win by being useful, transparent, and safe. Features should stay in the bootstrapper/customization/diagnostics lane: no Roblox memory patching, no anti-cheat bypasses, no hidden FastFlag bypasses, and no binary tampering.

## Wave 1: Power Tools

- Water Tools hub for diagnostics, cleanup, folders, and repair actions.
- Diagnostics zip export with logs, state, settings, and system report.
- One-click system report copy for support.
- Download cache, log, and temporary file cleanup.
- FastFlag reset and ClientAppSettings quick open.
- Queue Roblox reinstall/repair.
- Import settings from existing Bloxstrap installs.

## Wave 2: Profiles

- Named profiles for competitive, cinematic, low-end, studio, and default Roblox launches.
- Per-profile FastFlag sets with diff preview before saving.
- Per-profile Discord RPC preferences.
- Per-profile mod preset toggles.
- Profile import/export as JSON.
- â€œSafe defaultsâ€ profile that removes risky/unrecognized custom FastFlags.

## Wave 3: Launch Experience

- Launch dashboard with recent experiences, server region history, and last launch result.
- Pre-launch health check for missing DLLs, broken install payloads, stale shortcuts, and locked Roblox files.
- Optional launch confirmation with profile selector.
- Roblox repair prompt when checksums or expected version files are missing.
- Better offline/error pages with actionable fix buttons.

## Wave 4: Observability

- In-game log tail viewer for local Roblox logs.
- Waterstrap event timeline with install/update/launch/mod events.
- Exportable support bundle with redaction controls.
- Network check for Roblox deployment endpoints.
- Update channel status view with current, latest, and cached versions.

## Wave 5: Customization

- Air-themed bootstrapper styles.
- Built-in theme gallery with preview.
- Custom icon pack manager.
- Cursor, sound, font, and texture preset gallery.
- Custom bootstrapper theme validator.
- Theme import/export.

## Wave 6: Maintenance

- Leftover file cleaner for old Bloxstrap/Fishstrap/Waterstrap installs.
- Shortcut repair wizard.
- Registry protocol repair for roblox/player/studio links.
- Version folder pruning with keep-latest count.
- Backup and restore for settings, mods, themes, and FastFlags.

## Wave 7: Safety

- FastFlag allowlist/denylist with warnings for sensitive names.
- â€œExplain this flagâ€ notes for known safe presets.
- Scan custom themes for unsupported or suspicious file references.
- Show clear warnings for any feature that changes local Roblox content files.
- Never add memory modification, injection, exploit helpers, or bypass systems.
